# Practicals Repository

This repository contains various practical assignments for the sppu university and projects across different topics covered during academic studies. Below is an overview of the folders available:

## Folders

1. **Advance Python Programming**: Includes advanced concepts and practical applications in Python programming.

2. **Computational Geometry Practical**: Contains programs and exercises related to computational geometry.

3. **Data Structure II**: Implementation of data structures and algorithms for intermediate-level coursework.

4. **Database Management System-II**: Assignments and projects related to advanced database management concepts.
