### How to Use the Documentation Page

1. **Open Locally**: Simply open `index.html` in your web browser
2. **GitHub Pages**: The page can be deployed to GitHub Pages for online access
3. **Navigate**: Click on folder names to expand and see files
4. **View Code**: Click on any file to view its contents
5. **Download**: Use the download button to save files locally

### Features

- **Zero Backend Required**: All processing happens in the browser
- **Portable**: Single HTML file with embedded CSS and JavaScript
- **Real-time Updates**: Fetches latest content from GitHub repository
- **Support for Multiple File Types**: C, Python, Jupyter notebooks, CSV, PDF, and more
