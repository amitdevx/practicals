# Slip 15 — Computer Networks Solution Guide

## Q1: Short Answer Questions

### a) What is password security?

**Password Security Measures:**
- Minimum 12 characters
- Mix of uppercase, lowercase, digits, symbols
- Avoid dictionary words
- Unique per account
- Regular changes
- Multi-factor authentication

---

### b) What is cipher?

**Cipher:** Encryption technique to transform plain text to cipher text.

**Types:**
- Caesar Cipher: Shift each character
- Substitution: Replace characters
- Transposition: Rearrange characters
- Modern: AES, RSA, DES

---

### c) What are digital signatures?

**Digital Signatures:**
- Authenticate message sender
- Ensure message integrity
- Non-repudiation
- Uses private key to sign
- Public key to verify

**Algorithm:** RSA, DSA

---

### d) What is cryptography?

**Cryptography:**
- Science of secure communication
- Encryption/Decryption
- Authentication
- Integrity assurance

**Fields:**
- Symmetric: Single key
- Asymmetric: Public/private keys
- Hashing: One-way transformation

---

### f) What is network attack?

**Common Attacks:**
- DDoS: Flood attacks
- Man-in-the-Middle: Interception
- Phishing: Social engineering
- SQL Injection: Database exploitation

---

### g) What is data integrity?

**Integrity Measures:**
- Checksums
- Hash functions (MD5, SHA)
- Digital signatures
- Error detection codes
