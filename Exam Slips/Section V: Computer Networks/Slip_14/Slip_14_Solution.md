# Slip 14 — Computer Networks Solution Guide

## Q1: Short Answer Questions

### a) What is IPv6 addressing?

**IPv6 Features:**
- 128-bit addresses (2^128)
- Hexadecimal notation
- Built-in security (IPsec)
- No NAT needed
- Simpler header

**IPv6 Format:** 2001:0db8:85a3:0000:0000:8a2e:0370:7334

---

### b) How is address allocation done?

**Methods:**
- Static: Manual configuration
- DHCP: Automatic assignment
- DHCPv6: IPv6 dynamic allocation
- SLAAC: Stateless autoconfiguration

---

### c) What are subnetting formulas?

**Formulas:**
- Subnets = 2^borrowed_bits
- Hosts_per_subnet = 2^host_bits - 2
- Subnet_size = 256 / number_of_subnets
- Block_size = 2^host_bits

---

### d) What are network devices?

**Devices:**
- Router: Inter-network routing
- Switch: Local switching (L2)
- Gateway: Protocol conversion
- Firewall: Security
- Load Balancer: Traffic distribution

---

### f) What is Cisco IOS?

**IOS (Internetwork Operating System):**
- Cisco router operating system
- CLI interface
- Command modes: User, Privileged, Config
- Supports routing, switching, security

---

### g) What is network topology?

**Topologies:**
- Bus: Linear
- Star: Central hub
- Ring: Circular
- Mesh: Multiple connections
- Hybrid: Combination
