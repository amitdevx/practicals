# Slip 13 — Computer Networks Solution Guide

## Q1: Short Answer Questions

### a) What are networking standards?

**Standards Organizations:**
- IEEE (Institute of Electrical and Electronics Engineers)
- ISO (International Organization for Standardization)
- IETF (Internet Engineering Task Force)
- ITU (International Telecommunication Union)

**Important Standards:**
- IEEE 802.3: Ethernet
- IEEE 802.11: WiFi
- RFC standards: Internet protocols

---

### b) What is VPN?

**VPN (Virtual Private Network):**
- Encrypts internet traffic
- Hides IP address
- Creates secure tunnel
- Allows remote access securely

**Protocols:**
- IPsec
- SSL/TLS
- L2TP

---

### c) What is network monitoring?

**Monitoring Tools:**
- Wireshark: Packet analysis
- Nagios: Network monitoring
- SNMP: Simple Network Management Protocol
- Netflow: Traffic analysis

---

### d) How to trace packets?

**Tools:**
- Tcpdump: Capture packets
- Wireshark: Analyze captured packets
- Traceroute: Trace packet path

---

### f) What is protocol stack?

**Stack Layers:**
- Physical
- Data Link
- Network
- Transport
- Application

Each layer has specific protocols and functions.

---

### g) What is network redundancy?

**Redundancy Techniques:**
- Backup links
- Load balancing
- Failover mechanisms
- Mirrored devices
