## Important Commands Quick Reference

### Linux
```bash
# IP Configuration
ip addr show                    # Show IP addresses
ifconfig                        # Show interface config
sudo ip addr add IP/MASK dev IFACE   # Assign static IP
sudo dhclient -r                # Release DHCP
sudo dhclient                   # Request new DHCP

# Network Tools
ping IP                         # Test connectivity
traceroute IP                   # Trace route
nslookup DOMAIN                 # DNS lookup
netstat -an                     # Show connections
nmap -sV IP                     # Port scan
```

### Cisco IOS
```
enable                          # Enter privileged mode
configure terminal              # Enter config mode
hostname NAME                   # Set hostname
enable password PASS            # Set enable password
enable secret PASS              # Set encrypted password
line console 0 / password       # Console password
line vty 0 4 / password         # VTY password
interface g0/0                  # Select interface
ip address IP MASK              # Assign IP
no shutdown                     # Enable interface
show running-config             # View config
copy running-config startup-config  # Save config
```
